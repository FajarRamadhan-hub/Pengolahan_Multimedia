### Learning Outcomes

1. **Audio Processing**  
      Students are able to implement audio processing techniques using Python, including loading, analyzing, and transforming audio signals for feature extraction and simple applications.

2. **Image Processing & Facial Detection**  
      Students are able to design and implement an image processing pipeline, applying preprocessing, filtering, and feature extraction, as well as developing facial detection systems using Python libraries.

3. **Video Processing & Landmarking**  
      Students are able to develop video processing pipelines that incorporate pose estimation and hand landmark detection, and evaluate their performance in practical applications.
